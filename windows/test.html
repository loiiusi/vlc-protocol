VLC Protocol Handler Test

1. Standard Format

Basic format: vlc://[URL]
vlc://http://localhost:19798/static/http/localhost:19798/False/%2F115%2F123%2Ftest.mkv

2. Chrome 130+ Format

Chrome 130+ format (missing colon): vlc://http//[URL]
vlc://http//localhost:19798/static/http/localhost:19798/False/%2F115%2F123%2Ftest.mkv

3. Weblink Format

Weblink format: vlc://weblink?url=[URL]
vlc://weblink?url=http://localhost:19798/static/http/localhost:19798/False/%2F115%2F123%2Ftest.mkv

4. URL-Encoded Weblink Format

URL-encoded weblink format: vlc://weblink?url=[Encoded-URL]
vlc://weblink?url=http%3A%2F%2Flocalhost%3A19798%2Fstatic%2Fhttp%2Flocalhost%3A19798%2FFalse%2F%252F115%252F123%252Ftest.mkv
